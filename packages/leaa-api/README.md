# Leaa Api

## **ABOUT**

leaa-api base [Nest.js](https://github.com/nestjs/nest) framework.

## **INSTALLATION**

```shell script
# .env for development
cp .env.example .env

# .env for production
cp .env.example .env.production # for production

# run mysql with docker-compose
docker-compose down && docker-compose up -d

# init
yarn insatll && yarn seed --nuke

# dev
yarn dev
```
