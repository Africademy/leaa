# Leaa Api

## **ABOUT**

leaa-api base [Nest.js](https://github.com/nestjs/nest) framework.

## **INSTALLATION**

```shell script
docker-compose down && docker-compose up -d

# .env for development
cp .env.example .env

# .env for production
cp .env.example .env.production # for production

yarn install
yarn dev
```
