# Leaa WWW (Website)

## **ABOUT**

leaa-www base [Next.js](https://nextjs.org/) framework.

## **INSTALLATION**

```shell script
# .env for development
cp .env.example .env

# .env for production
cp .env.example .env.production # for production

yarn install
yarn dev
```

## **MEMO**

in `package.json`, `css-loader` MUST KEEP  `1.0.1`.
